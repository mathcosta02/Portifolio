Projeto 01 TCC

Sistema de detecção de potenciais criadouros de mosquito Aedes aegypti

Eu e meu grupo fizemos um sistema de solução integrada que utiliza inteligência artificial e participação da população para identificar possíveis criadouros do mosquito Aedes aegypti: por meio de um aplicativo móvel, os usuários capturam e enviam imagens de ambientes como quintais e terrenos, que são processadas em um backend onde um modelo de visão computacional detecta objetos com potencial de acúmulo de água; após a análise, os resultados são armazenados e apresentados tanto ao usuário, com marcações nas imagens, quanto a gestores de saúde em um portal web com mapa interativo georreferenciado, permitindo visualizar áreas de maior risco e apoiar ações preventivas mais rápidas e eficientes.

  • HTML - CSS - JS
  • React Native
  • AI

Projeto 03 DASHBOARD

Dashboard analise de produtos

Nesse projeto peguei os dados de uma loja no mercado livre que são armazenados no sistema Bling e fiz uma integração com o Make para que informações específicas sejam enviadas ao google planilhas. A partir desses dados o sistema que ainda está em desenvolvimento será capaz de em um dashboard mostras informações importantes ao cliente de todos os pedidos já concluidos e todos os pedidos que serão feitos.
